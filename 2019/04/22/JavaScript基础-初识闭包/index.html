什么是闭包？

闭包其实是一种函数，所以，闭包技术也是函数的一种；闭包能做的事函数都能做。（定义在一个函数内部的函数）
当一个内部函数被其外部函数之外的变量引用时，就形成了闭包。

闭包的用处：

  • 一是读取函数内部的变量；
  • 二是让这些变量始终储存在内存中。

JavaScript的GC机制

在JavaScript中，如果一个对象不再被引用，那么这个对象就会被GC回收，否则这个对象一直保存在内存中。
